When you run the code, it will begin my "life", after that has finished running, the code will execute what's called a recursive loop. What this does is that it will replicate my code (aka my life) and the 3 stories within my life. Then that code will duplicate it self, and so on and so forth. This is an abstract representation of my influence and my legacy. This is how I hope how my life will end. That a bit of me still permeates through this existence.

However, there's a catch. This program will run forever until the it breaks the loop. The loop is broken when the algorithm reaches the year 71,980 A.D... approx. 69,000 years later. When that day comes, the algorithm will delete itself. Thus representing the finitude of my existence. Even my immortal machine is not 100% immortal.
